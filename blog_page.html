  • Q.1. Write the difference between map, filter, find & forEach.

    Answer : Map takes an array and returns an array by default.

    In similar case when ForEach takes an array it returns undefined by default.

    filter checks the given condition and returns an array of all the matched objects.

    Find does the same task as filter, but unlike filter it returns the first matched object.

  • Q.2. Write the difference between localStorage, sessionStorage & Cookies.

    Answer : localStorage Doesnt deletes saved data even if the pc/browser is closed.

    sessionStorage deletes the data once the current tab is closed.

    Cookies can carry a expiration date.

  • Q.3. What is Eventloop? How does it work?

    Answer : As Javascript is synchronous, Event loop allows Js to run asynchronously. So when from a set of called functions, suppose the middle one has setTimeOut set to 4 seconds. So the function with setTimeOut will be executed at last. As the first normal statement enters the callstack and executes immediately. Then suppose the setTimeOut function is the second one. As its a asynchronous function, it get pushed into the event table and then into the event queue after a certain time. In the mean time all the normal statements gets executed. lastly setTimeOut function moves to callStack and gets executed.
